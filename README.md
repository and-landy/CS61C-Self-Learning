# CS61C Self-Learning

A personal repository documenting my self-study of **UC Berkeley's CS61C** (Great Ideas in Computer Architecture), completed alongside my ECE coursework at Rutgers University.

## Background

I'm an Electrical and Computer Engineering student at Rutgers University, self-studying CS61C to strengthen my foundation in computer systems — covering C, RISC-V assembly, CPU design, and parallel computing. This repo contains my solutions and notes for the labs and projects.

## Contents

- **Labs** — Hands-on exercises covering C programming, memory management, RISC-V assembly, and more
- **Projects** — Larger assignments including:
  - `Project 1` — C programming & bitmap manipulation
  - `Project 2` — RISC-V assembly (MNIST neural network inference)
  - `Project 3` — CPU design (in progress)
  - `Project 4` — Parallelism & performance optimization (in progress)

## Note

This repository is intended as a **personal reference**. If you're taking CS61C, I encourage you to attempt the assignments yourself first — the struggle is where the learning happens.
